
The home page for users to visit to navigate through the site is
home.php.
This page allows users to search for products and order products.
The cart.php file allows users to view their order details and remove
products they do not want to purchase anymore.

The admin.html file is the home page for the administrator. From this
page you can navigate to other pages to add employees and search for both
customers and employees. From this page, the administrator can also add 
products and edit them.
